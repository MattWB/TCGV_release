# Contribuer

Merci pour votre contribution.

## Prerequis

- Node.js `20.18.0` (voir `.nvmrc`)
- npm

## Workflow

1. Creer une branche depuis `main`
2. Faire des commits atomiques et explicites
3. Lancer les commandes locales avant PR:
   - `npm install`
   - `npm run dev`
   - `npm run build`
   - `npm run format` (si necessaire)
4. Ouvrir une PR avec:
   - Contexte
   - Changements
   - Verification manuelle

## Style

- Respecter `.editorconfig`
- Eviter les changements hors scope
- Ne pas introduire de secrets dans le repo
